## Instalation

- clone repository
- `yarn install`
- `cd ios && pod install` (just for iOS)
- before running the apps If you have already installed JDK on your system, make sure it is JDK 11 or newer.
- `yarn android` (for Running on Android)
- `yarn ios` (for Running on ios)
